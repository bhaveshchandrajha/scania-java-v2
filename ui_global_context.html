1. Inputs & Global Context

Point to your AST directory (with *-ast.json files) and the RPG root directory (e.g. extracted PoC_HS1210). Then click Build Global Context. Afterwards you can export a Neo4j Cypher script in step 2.


      

    


    

      

        
2. Knowledge Graph

3. Migrate Feature

        
Select a program and a starting point (procedure or subroutine), then migrate that feature using the
